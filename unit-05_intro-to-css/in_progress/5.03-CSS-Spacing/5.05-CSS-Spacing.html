CSS Spacing Examples

No Spacing Example

This section has no margin or padding. Notice how cramped it feels and how the text goes right to the edges. This is hard to read and doesn't look professional.

Good Spacing Example

This section has margin and padding. Notice how much more comfortable this feels to read. The content has room to breathe and feels more organized.

This Box Has Padding and Margin

This gray box demonstrates both padding (space inside the border) and margin (space outside the border). The padding keeps the text away from the border, and the margin keeps this box away from other elements.

Another Box for Comparison

Notice the space between this box and the one above. That's the margin working! The space inside this box (between the text and border) is the padding.

Website Header Example

This header section uses padding to create space around the content and margin-bottom to separate it from the next section.

Main Content Section

This content section has padding for internal spacing and margin for separation from other elements. The border and background color help define the section boundaries.

Multiple paragraphs within the same section benefit from the consistent internal padding.

Another Content Section

Notice how the margin between sections creates clear separation. Each section feels distinct but part of the same page.

Sidebar-Style Section

This section uses different spacing and a colored border to create a sidebar effect. The margin-left and padding work together to create this layout.
