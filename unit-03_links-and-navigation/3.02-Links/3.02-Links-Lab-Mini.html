HTML Links Lab: Connecting Pages

Welcome to your first link exercise! Links let users jump to different pages. We use the <a> tag for every link.

Part 1: The Main Page (index.html)

In your first file, make two links:

  • A New Tab Link (External): Make a link to a website like Google or Wikipedia. Use the target="_blank" rule so it opens in a new tab.
  • A Second Page Link (Internal): Make a link to a page you will create next. Name the file other-page.html.

Part 2: The Second Page (other-page.html)

Now, create the second file and make one link:

  • A Home Link (Internal): Make a link that goes back to your first file (index.html).

Quick Tips

Remember:

  • Use href to tell the link where to go.
  • Full web address (like https://...) for links outside your website.
  • Just the filename (like other-page.html) for links inside your website.
