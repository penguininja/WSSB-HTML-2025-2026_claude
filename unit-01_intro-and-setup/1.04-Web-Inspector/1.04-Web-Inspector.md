CHROME DEVELOPER TOOLS CHEAT SHEET

OPENING DEVELOPER TOOLS:
F12 - Opens Developer Tools

BASIC NAVIGATION:
F6 - Move between panels (Elements, Console, etc.)
Tab - Move forward within a panel
Shift+Tab - Move backward within a panel
Escape - Close Developer Tools

ELEMENT INSPECTOR:
Arrow Keys - Navigate up/down through HTML elements
Right Arrow - Expand (open) an element to see what's inside
Left Arrow - Collapse (close) an element

Insert+Z - Read current line
Insert+F5 - Refresh what JAWS can see
